Sistemas Implementados

Cada sistema funciona end-to-end, probado con tests de integracion automatizados.

⚔

Combate en Tiempo Real

Formula de dano dual (fisico/magico) con accuracy, evasion, criticos, mitigacion por defensa, y condiciones de combate con duracion.

✨

Sistema de Skills

6 habilidades + ultimate por clase. Cada skill tiene costos de mana, cooldowns, tipos de dano, y condiciones aplicables. Ascension modifica los valores.

🌐

Mundo Multiplayer

Mundo compartido 100x100 con posiciones en tiempo real. Movimiento cardinal, click-to-move, visibilidad por radio, y sincronizacion via SignalR WebSockets.

💬

Sistema Social

Chat local por proximidad, solicitudes de amistad, lista de bloqueados, perfiles publicos. Validacion por rango de interaccion real.

📈

Progresion Profunda

50 niveles, 10 niveles de ascension, curvas de XP, stats derivados, Power Score compuesto. Cada decision de build impacta el combate.

📱

Cliente Movil (Flutter)

App Android nativa con joystick virtual, canvas de mundo, barras de skills, chat, y sistema social. Misma conexion SignalR que el cliente web.
